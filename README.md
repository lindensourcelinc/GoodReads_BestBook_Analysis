# GoodReads_BestBook_Analysis

A Python data analysis project utilizing Jupyter Notebooks.

## Setup Instructions

Follow these steps to set up your local environment and run the notebook:

### 1. Clone the Repository
```bash
git clone <https://github.com/lindensourcelinc/GoodReads_BestBook_Analysis.git>
cd keystone
```

### 2. Create a Virtual Environment
Create your own local virtual environment (this folder is ignored by Git):
```bash
python -m venv .venv
```

### 3. Activate the Environment
* **Mac / Linux:**
  ```bash
  source .venv/bin/activate
  ```
* **Windows (Command Prompt):**
  ```cmd
  .venv\Scripts\activate.bat
  ```
* **Windows (PowerShell):**
  ```powershell
  .venv\Scripts\Activate.ps1
  ```

### 4. Install Dependencies
Install all required libraries listed in `requirements.txt`:
```bash
pip install -r requirements.txt
```

### 5. Run the Notebook
Open your code editor (like VS Code), open `notebooks/notes.ipynb`, and ensure your Jupyter kernel is set to **Python (.venv)**.
